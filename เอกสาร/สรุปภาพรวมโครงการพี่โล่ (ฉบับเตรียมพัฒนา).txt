🛡️ สรุปแผนงานโครงการ "พี่โล่" (Pi Loh)
1. วิสัยทัศน์ (Core Vision)
"พี่โล่" จะเป็นโปรแกรมติดตั้งในเครื่อง (Desktop App) ที่ทำหน้าที่เป็น ผู้ช่วยบัญชีและที่ปรึกษาภาษีส่วนตัว โดยชูจุดเด่นเรื่อง "ความเป็นส่วนตัว 100%" ข้อมูลทุกอย่างประมวลผลในเครื่องลูกค้า ไม่ส่งขึ้น Cloud เหมาะกับ SME ไทยที่กังวลเรื่องความลับทางการเงิน
2. สถาปัตยกรรมทางเทคนิค (Technical Stack)
เราจะใช้การพัฒนาแบบ Hybrid (Mac for Windows):
* Development Platform: พัฒนาและทดสอบบน Mac (โดยใช้ Antigravity ช่วยเขียนโค้ด)
* Target Platform: ผลลัพธ์สุดท้ายคือไฟล์ .exe สำหรับ Windows (แจกจ่ายแบบ Standalone ไม่ผ่าน Store)
* Frontend: React / Tailwind CSS (อิงตาม pi_loh_app.html)
* Backend Wrapper: Tauri (Rust) เพื่อความเบาและเข้าถึงไฟล์ในเครื่องได้ปลอดภัย
* AI Engine: ZeroClaw รันแบบ Sidecar (ฝังไปกับโปรแกรม)
* Brain (LLM): ใช้โมเดลภาษาไทยเฉพาะทาง (เช่น Typhoon) ในรูปแบบ GGUF เพื่อรันแบบ Offline
3. แผนการดำเนินงาน Phase 1: AI Consultant
เราจะเริ่มจากสิ่งที่ทำได้เร็วและเห็นผลชัดที่สุดคือ "ระบบแชทที่ปรึกษา":
* หน้าที่: ตอบคำถามภาษี บัญชี และข้อกฎหมายไทย
* เทคนิค: ใช้ระบบ RAG (Retrieval-Augmented Generation) คือการเอา "ประมวลรัษฎากร" และ "คู่มือบัญชี" มาทำเป็นฐานข้อมูลให้ AI เปิดตำราตอบ
* User Experience: หน้าแชทที่สวยงาม มีปุ่มทางลัดคำถามยอดฮิต และแสดงมาตราอ้างอิงทุกครั้งที่ตอบ
4. กลยุทธ์การแจกจ่าย (Distribution Strategy)
* Zero-Cost: ไม่เน้นการซื้อใบรับรองราคาแพงในระยะแรก เน้นสร้างความเชื่อใจผ่านผลลัพธ์ของโปรแกรม
* Standalone .exe: ลูกค้าดาวน์โหลดไปติดตั้งเองได้ทันที
* First-run Setup: โปรแกรมจะจัดการติดตั้ง "สมอง AI" ให้เองเมื่อเปิดใช้งานครั้งแรก เพื่อให้ผู้ใช้ทั่วไปใช้งานง่ายที่สุด
